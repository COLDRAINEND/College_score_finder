# College_score_finder
When  bored during the winter vacation, A score query developed by python, mainly using the framework of web crawler, BS4 and Qt5. Package with pyinstaller.


    １．　批量查询时将学号文件放在本程序根目录，并命名为　xuehao.txt。　学号文件中的格式为：　<学号><四个空各><姓名>
    ２．　xuehao.txt 具体文件请看 本项目中的例子。
    ３．　计算机中需安装python3 以及，BS4 界面解析库。
